##### 1如何提交git的文件到暂存区

##### git add 文件名

```
git add *.c 这意味着将所有.c结尾的文件加入到暂存区中去
```



##### 2如何提交文件到完成区

git commit -m ‘ 内容‘



3将处于完成区的图片同步到github中：

4删除文件：

​			git rm -r 文件夹名

6什么是fork
